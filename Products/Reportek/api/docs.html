get /envelopes
(envelopesGet)

Query parameters

isReleased (optional)
Query Parameter — Released(1) or Unreleased(0) envelopes to return
url (optional)
Query Parameter — Url of the envelopes or part of to return
countryCode (optional)
Query Parameter — Country code in ISO \"ALPHA-2 Code format for envelopes to return
reportingDate (optional)
Query Parameter — Return envelopes with this reporting date. ISO8601(YYYY-MM-DD) date format.
reportingDateStart (optional)
Query Parameter — Return envelopes that have a reporting date in range of reportingDateStart and reportingDateEnd or today if reportingDateEnd is missing. ISO8601(YYYY-MM-DD) date format.
reportingDateEnd (optional)
Query Parameter — Return envelopes that have a reporting date in range of reportingDateStart and reportingDateEnd. If reportingDateStart is missing, all envelopes with reporting date <= reportingDateEnd will be returned. ISO8601(YYYY-MM-DD) date format.
obligations (optional)
Query Parameter — Return envelopes with one of the following obligations
periodDescription (optional)
Query Parameter — Return envelopes with specified periodDescription
modifiedDate (optional)
Query Parameter — Return envelopes with specified modifiedDate. ISO8601(YYYY-MM-DD) date format.
modifiedDateStart (optional)
Query Parameter — Return envelopes that have a modified date in range of modifiedDateStart and modifiedDateEnd or today if modifiedDateEnd is missing. ISO8601(YYYY-MM-DD) date format.
modifiedDateEnd (optional)
Query Parameter — Return envelopes that have a modified date in range of modifiedDateStart and modifiedDateEnd. If modifiedDateStart is missing, all envelopes with modified date <= modifiedDateEnd will be returned. ISO8601(YYYY-MM-DD) date format.
status (optional)
Query Parameter — Return envelopes that have the following status
statusDate (optional)
Query Parameter — Return envelopes that have a status with the following statusDate. ISO8601(YYYY-MM-DD) date format.

Return type

array[EnvelopeDefaultAttributes]
